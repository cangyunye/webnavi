# 资源导航网站规划方案

## 项目概述

* **技术栈**: Python 3.12 + FastAPI + MySQL

* **项目名称**: Navigator

* **核心功能**: 资源导航网站，支持分类展示和折叠式侧边栏

***

## 二、数据库设计

### 2.1 分类表 (categories)

| 字段          | 类型              | 说明   |
| ----------- | --------------- | ---- |
| id          | INT PRIMARY KEY | 分类ID |
| name        | VARCHAR(50)     | 分类名称 |
| icon        | VARCHAR(100)    | 图标类名 |
| sort\_order | INT             | 排序序号 |

**预设数据**:

* 学习 (study)

* 数据库 (database)

* 研发机器 (dev-machine)

* AI (ai)

* 移动通信 (mobile)

* 软件资源 (software)

* 运维 (operations)

### 2.2 资源表 (resources)

| 字段           | 类型              | 说明            |
| ------------ | --------------- | ------------- |
| id           | INT PRIMARY KEY | 资源ID          |
| category\_id | INT             | 关联分类ID        |
| name         | VARCHAR(100)    | 资源名称          |
| url          | VARCHAR(500)    | 链接地址          |
| description  | TEXT            | 描述            |
| status       | TINYINT         | 状态(0=离线,1=在线) |
| create\_time | DATETIME        | 创建时间          |
| update\_time | DATETIME        | 更新时间          |

### 2.3 研发机器表 (dev\_machines)

| 字段           | 类型              | 说明                |
| ------------ | --------------- | ----------------- |
| id           | INT PRIMARY KEY | 机器ID              |
| name         | VARCHAR(100)    | 机器名称              |
| ip           | VARCHAR(50)     | IP地址              |
| port         | INT             | 端口                |
| hostname     | VARCHAR(100)    | 主机名               |
| cpu          | VARCHAR(50)     | CPU信息             |
| memory       | VARCHAR(50)     | 内存信息              |
| disk         | VARCHAR(100)    | 磁盘信息              |
| os           | VARCHAR(100)    | 操作系统              |
| status       | TINYINT         | 状态(0=离线,1=在线)     |
| environment  | VARCHAR(50)     | 环境(dev/test/prod) |
| description  | TEXT            | 描述                |
| create\_time | DATETIME        | 创建时间              |

### 2.4 数据库实例表 (db\_instances)

| 字段           | 类型              | 说明                       |
| ------------ | --------------- | ------------------------ |
| id           | INT PRIMARY KEY | 实例ID                     |
| name         | VARCHAR(100)    | 实例名称                     |
| db\_type     | VARCHAR(50)     | 数据库类型(MySQL/PostgreSQL等) |
| version      | VARCHAR(50)     | 版本                       |
| ip           | VARCHAR(50)     | 主机IP                     |
| port         | INT             | 端口                       |
| charset      | VARCHAR(20)     | 字符集                      |
| status       | TINYINT         | 状态(0=离线,1=在线)            |
| environment  | VARCHAR(50)     | 环境                       |
| description  | TEXT            | 描述                       |
| create\_time | DATETIME        | 创建时间                     |

## 三、API 接口设计

### 3.1 分类接口

* `GET /api/categories` - 获取所有分类

### 3.2 研发机器接口

* `GET /api/dev-machines` - 获取研发机器列表

* `GET /api/dev-machines/{id}` - 获取单个机器详情

### 3.3 数据库实例接口

* `GET /api/db-instances` - 获取数据库实例列表

* `GET /api/db-instances/{id}` - 获取单个实例详情

## 四、前端页面设计

### 4.1 布局结构

```
+------------------------------------------+
|  [Logo]     ResourceNav                  |
+---+--------------------------------------+
|   |                                      |
| N |         主内容区域                    |
| A |    (显示选中分类的资源列表)           |
| V |                                      |
|   |                                      |
| [=]|                                      |
+---+--------------------------------------+
```

### 4.2 侧边栏设计

**展开状态**:

* 宽度: 200px

* 显示分类名称

* 左侧有颜色标识条

**收缩状态**:

* 宽度: 60px

* 仅显示图标

* 悬停显示分类名称

### 4.3 资源列表展示 (运维模板风格)

**研发机器列表字段**:

* 名称 | IP | 环境 | 状态 | CPU | 内存 | 操作系统

**数据库实例列表字段**:

* 名称 | 类型 | 版本 | IP:端口 | 环境 | 状态 | 字符集

### 4.4 样式风格

* **主色调**: 浅灰 #F5F7FA

* **侧边栏**: 白色 #FFFFFF，左侧彩条 #3B82F6

* **文字色**: #333333, #666666

* **状态指示**: 在线绿色 #22C55E, 离线灰色 #9CA3AF

* **边框**: #E5E7EB

* **圆角**: 8px

* **阴影**: 0 2px 8px rgba(0,0,0,0.05)
