# 右侧展开式添加表单实现计划

## 需求说明

将研发机器和数据库的添加按钮从表格头部移动到右侧操作列，点击后通过右侧滑出面板展开表单进行填写。

## 实现内容

### 1. CSS 样式 (frontend/css/style.css)

新增右侧滑出面板样式：
- `.side-panel` - 右侧滑出面板容器
- `.side-panel.active` - 展开状态
- `.side-panel-content` - 面板内容区
- 过渡动画效果

### 2. HTML 结构 (frontend/index.html)

在 body 末尾添加右侧滑出面板容器：
```html
<div id="side-panel" class="side-panel">
    <div class="side-panel-header">
        <h2 id="side-panel-title">添加研发机器</h2>
        <button class="close-btn" onclick="closeSidePanel()">×</button>
    </div>
    <div id="side-panel-body" class="side-panel-body">
        <!-- 表单内容 -->
    </div>
</div>
```

### 3. JavaScript (frontend/js/app.js)

更新 `renderDevMachinesTable` 和 `renderDbInstancesTable`：
- 移除 header-actions 中的添加按钮
- 在操作列添加「添加」按钮
- 点击按钮时调用 `openAddMachinePanel()` 或 `openAddDbPanel()`

新增函数：
- `openSidePanel(title, content)` - 打开右侧面板
- `closeSidePanel()` - 关闭右侧面板
- `openAddMachinePanel()` - 打开研发机器添加面板
- `openAddDbPanel()` - 打开数据库添加面板

更新添加函数：
- `addDevMachine()` - 添加成功后关闭右侧面板
- `addDbInstance()` - 添加成功后关闭右侧面板

## 实现步骤

1. 在 index.html 添加右侧滑出面板 HTML 结构
2. 在 style.css 添加滑出面板样式
3. 更新 app.js 中的表格渲染函数，将添加按钮移到操作列
4. 实现右侧面板的打开/关闭函数
5. 实现研发机器和数据库的右侧面板添加表单
6. 更新添加成功后关闭面板的逻辑
