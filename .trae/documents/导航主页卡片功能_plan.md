# 导航主页卡片功能实现计划

## 功能概述
在资源导航网站的主页添加卡片式分类展示，点击卡片可以跳转到对应分类。

## 当前代码分析

### 现有结构
- 主页使用 `welcome-panel` 显示简单欢迎信息
- 左侧导航栏已有 7 个分类
- 点击左侧导航可以切换到对应分类内容

### 需要修改的文件
1. `frontend/js/app.js` - 添加主页渲染和跳转逻辑
2. `frontend/css/style.css` - 添加卡片样式
3. `frontend/index.html` - 添加logo点击返回主页功能

## 实现步骤

### 1. 更新 `app.js`
- 添加 `renderHomePage()` 函数，用于渲染主页卡片
- 添加 `goHome()` 函数，用于返回主页
- 修改 `DOMContentLoaded` 事件，初始化显示主页
- 添加点击卡片触发分类跳转的逻辑

### 2. 更新 `style.css`
- 添加 `.home-container` 样式 - 主页容器
- 添加 `.card-grid` 样式 - 卡片网格布局
- 添加 `.category-card` 样式 - 单个分类卡片
- 添加卡片悬停动画效果
- 添加卡片中的图片区域样式
- 保持与现有浅色调风格一致

### 3. 更新 `index.html`
- 为 logo 区域添加点击事件，点击返回主页

## 设计细节

### 卡片结构
```
+-------------------+
|                   |
|    图标区域       |
|                   |
+-------------------+
|   分类名称        |
+-------------------+
```

### 样式要点
- 卡片使用白色背景，带轻微阴影
- 圆角设计，悬停时轻微上浮
- 图标区域使用渐变色背景
- 响应式网格布局，适配不同屏幕

### 分类图片配置
使用 emoji 图标或纯色背景 + 文字标识，保持风格一致。

## 文件修改清单

1. **frontend/js/app.js**
   - 添加 `renderHomePage()`
   - 添加 `goHome()`
   - 修改初始化逻辑

2. **frontend/css/style.css**
   - 添加卡片相关样式

3. **frontend/index.html**
   - 为 logo 区域添加点击事件

## 预期效果
- 打开网站首先看到 7 个分类卡片
- 点击卡片跳转到对应分类
- 点击 logo 返回主页
- 保持现有导航功能不变
